    This is a tiny URL shortener service
    Written in JavaScript using Node.js
    Node.js must be installed on the local machine
    To start it, just type in the Windows command line
       node app.js
    To connect to the service, type in the browser command line:
       http://localhost:9999
    Then type short URL in the text field. Request will be redirected to a proper URL
    URL database is the text file 'database.dat'
